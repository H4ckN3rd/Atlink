# **App Name**: AtLink Dashboard

## Core Features:

- Attendance Metrics Dashboard: Display key attendance metrics (Present, Absent, Total Users, Attendance Rate) fetched from the Firebase Realtime Database via API calls, presented in customizable Shadcn UI Card components.
- Data Fetching and Error Handling: Implement robust data fetching mechanisms to consume attendance and user data from the Firebase Realtime Database, managing loading states and errors gracefully.
- Interactive Attendance Table: Create a dynamic Shadcn UI Table component featuring pagination, column sorting, and toggleable column visibility for comprehensive data review.
- Advanced Filtering System: Develop an advanced filtering interface using Shadcn UI components like DateRangePicker, Select, and Combobox for detailed data segmentation by date, users, and attendance status.
- Analytics Graph Generation: Generate insightful visualizations of attendance trends, user-specific patterns, and hourly distributions using a charting tool with responsive capabilities.
- Dark/Light Theme Toggle: Integrate a seamless dark/light theme toggle powered by Shadcn UI's theming, enhancing user experience based on preference.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) to evoke a sense of trust and reliability.
- Background color: Light gray (#F9FAFB), desaturated for a clean, modern look.
- Accent color: Vibrant purple (#A855F7) for interactive elements and key highlights.
- Body and headline font: 'Inter' (sans-serif) for a clean and modern user interface.
- Crisp, modern line icons for visual clarity and enhanced user understanding.
- Clean, structured layout using grids for balanced content distribution and responsiveness.
- Subtle transitions and animations to provide feedback on interactions, enhancing the overall user experience without distraction.
